NEXUS DFS — Premium Depth-First Search Visualizer

        
A breathtaking, fully interactive demonstration of the iterative Depth-First Search algorithm. Designed to stop recruiters, clients, and fellow engineers in their tracks.


        
Features

        
    
          
  • Real-time animated canvas traversal with live stack visualization
    • 
          
  • Exact parity with all six lab test cases
    • 
          
  • Vanilla JS + Tailwind + Canvas (zero dependencies)
    • 
          
  • Cross-language implementations (JS, Python 3, Java 17+)
    • 
          
  • Glassmorphic cyber-luxury UI with neon accents
    • 
          
  • Responsive down to mobile
    •

Just open index.html in any modern browser. No build step. No npm. Pure elegance.
